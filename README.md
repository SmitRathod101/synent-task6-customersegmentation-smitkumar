# Customer Segmentation using K-Means Clustering

## Project Overview

This project performs customer segmentation using the Mall Customer Dataset. K-Means Clustering is applied to group customers based on their annual income and spending behavior.

## Objective

- Perform exploratory data analysis (EDA)
- Identify customer groups using clustering
- Visualize customer segments
- Generate business insights

## Dataset

Dataset: Mall Customer Dataset

Features:
- CustomerID
- Genre
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Workflow

1. Data Loading
2. Data Exploration
3. Visualization
4. Elbow Method
5. K-Means Clustering
6. Customer Segmentation
7. Business Insights

## Results

The Elbow Method indicated that 5 clusters were optimal.

### Customer Segments

- High Income – High Spending Customers
- High Income – Low Spending Customers
- Low Income – High Spending Customers
- Low Income – Low Spending Customers
- Average Customers
